# Futured iOS Fastlane

Reusable [Fastlane](https://fastlane.tools) configuration for iOS CI/CD. Provides lanes for provisioning, testing, beta/release builds, and App Store Connect management — designed to be imported via `import_from_git` by organization adapters or directly by individual projects.

## Documentation

Full documentation is available at **[futuredapp.github.io/ios-fastlane](https://futuredapp.github.io/ios-fastlane)**.
